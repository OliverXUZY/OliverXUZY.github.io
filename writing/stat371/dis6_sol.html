1.2.2.1 Determine a 98% bootstrap t confidence interval

Determine a \(98\%\) bootstrap t confidence interval for the town-wide average pounds/week for citizens from this city (use set.seed(1)).

Click For Answer

Using the bootstrap code given below,

  • \(t_.99=-3.09\) and \(t_.01=2.31\) (using the convention that the subscript gives area above the value.).

  • The sample mean is \(\bar{x}=6.44\) and sample standard deviation is \(sd_X=3.24\).

  • So a \(98\%\) bootstrap t confidence interval for the town average pounds/week of recycling in their garbage is given by: \(L=6.44-2.31*(\frac{3.24}{\sqrt{14}})=4.44\) and \(U=6.44--3.09*(\frac{3.24}{\sqrt{14}})=9.12\) so (4.44, 9.12)

#Create Bootstrap Function
bootstrap = function(x, n.boot) {
  n = length(x)
  x.bar <- mean(x)
  t.hat <- numeric(n.boot) # create vector of length n.boot zeros
  for(i in 1:n.boot) {
    x.star <- sample(x, size=n, replace=TRUE)
    x.bar.star <- mean(x.star)
    s.star <- sd(x.star)
    t.hat[i] <- (x.bar.star - x.bar) / (s.star / sqrt(n))
}
  return(t.hat)
}

#Call Bootstrap Function on our data to get bootstrap T distribution
set.seed(1)
Recycling.boot <- bootstrap(recycling, 5000)
# Plot the approximate sampling distribution.

1.4 Comparison

Comment on the similarities and differences between the assumptions that go into these two confidence intervals and the resulting computed intervals. Which interval would you recommend the city to use?

Click For Answer

For the t confidence interval, we are assuming the sample comes from a normal population, which we are not making with the bootstrap T. We see in t interval is symmetric around the sample mean (ME same in each direction), which is not the case with the bootstrap t. It depends on the assumptions that the city wants to make. The qqnorm plot of the data gives some evidence of non-linearity (and the histogram shows lack of bell curve). The bootstrap is making fewer assumptions and is wider (and thus a bit more conservative); therefore I would tend to recommend the bootstrap in this case.

3 Problem 3: Confidence Interval concept review.

  1. This is the equation for building a \((1-\alpha)*100\%\) large-sample Z confidence interval for \(\pi\):

\[ \bbox[#ffcc99,20px,border:10px ligth red] { \hat{p}\pm z_{\alpha/2}*\sqrt{\frac{\hat{p}*(1-\hat{p})}{n}} } \]

  1. This is the equation for building a \((1-\alpha)*100\%\) t confidence interval for \(\mu\):

\[ \bbox[#ffcc99,20px,border:10px ligth red] { \bar{x} \pm t_{\alpha/2, df}*\frac{s}{\sqrt{n}} } \]

  1. This is the equation for building a \((1-\alpha)*100\%\) bootstrap t confidence interval for \(\mu\):

\[ \bbox[#ffcc99,20px,border:10px ligth red] { (\bar{x} - t^B_{\alpha/2}*\frac{s}{\sqrt{n}}, \bar{x} - t^B_{1-\alpha/2}*\frac{s}{\sqrt{n}}) } \]

A. When would we choose to use each equation?

Click For Answer

  • We should use (i) to estimate a population proportion of successes when I have an observed sample proportion of successes. A large enough sample size \(n\hat{p}\) and \(n(1-\hat{p})\) are both at least 5 so that the CLT tells me the distribution of \(\hat{p} \approx N\).

  • We should use (ii) to estimate a population mean value when I have an observed sample mean, a large enough sample size so that the CLT tells me the distribution of \(\bar{X} \approx N\) considering how non-normal I believe my population to be, and I am estimating my population sd with sample sd.

  • We should use (iii) to estimate a population mean value when I have an observed sample mean, and I do not want to make any normality assumptions about the population or CLT.

B. Identify the pieces of the equations that are the point estimates and which dictate the margin of error (half-width of the CI). What do the symbols in the equations stand for?

Click For Answer

  • Point estimates:\(\bar{x}\) sample mean, \(\hat{p}\): sample proportion

  • ME (quantity added/subtracted from p estimate)

  • t/z/t^B multipliers come from the level of the confidence interval desired and are values in the t/z/t^(boot) distribution that has \((1-\alpha)*100\%\) of values between them; value multiplied to t/z/t^B are the approximate Standard Errors for the point estimators, i.e.how much the \(\bar{x}\) and \(\hat{p}\) values vary between samples.

C. What factors in the confidence interval construction will result in the most precise estimate for our population parameters? (Can use http://www.rossmanchance.com/applets/ConfSim.html to see the effect of changing specific characteristics)

Click For Answer

The most precise estimate for the population parameter means the narrowest interval: lower confidence level, larger sample size, smaller sample standard deviation.
